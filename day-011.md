## pathlib
+ file path를 처리하기
+ 최근 여러 코드에서 자주 발견됨
+ python 기본 library

## how to use
+ get current path
+ path join
+ exist check
+ make directory
+ child list
+ `Path` to string
```python
from pathlib import Path

current_path = Path.cwd()
save_path = current_path / "results"
if not save_path.exists():
    save_path.mkdir(parents=True)
for child in current_path.iterdir():
    print(f"{child}")
```
